# SpringDataJPA 웹 프로젝트 연습


- Spring Data Jpa 공간입니다.  ' 코드로 배우는 스프링 부트 웹 프로젝트' 책을 참고하여 학습한 공간 입니다.

- 기본적인 CRUD 와 Spring Data JPA를 이용한 게시판 만들기 + 페이징 작업까지 한 프로젝트 입니다.

- 쿼리메서드,@Query, JPQL이 사용 되였습니다. Querydsl 은 사용되지 않았습니다.

## 개발환경
- JAVA
- IntelliJ +Gradle
- MariaDB
- SpringBoot 
- Thymeleaf
- Spring Data JPA
